（4）:nth-child(an + b) 的使用方式

使用公式 (an + b)。描述：a表示周期的长度，n 是计数器（从 0 开始），b 是偏移值。

例：p:nth-child(3n + 2) { color:red }     ------ 表示以3(a)个p标签为一个周期，将每个周期中的第2（b）个p标签的字体设置为红色
